# X METAL لوحة التحكم

مشروع ويب ثابت لواجهة لوحة تحكم متكاملة بحالة RTL وميزات متقدمة مثل مخزون المنتجات، سجل المبيعات، تحليلات، وإعدادات سلايدر.

## بنية المشروع

- `index.html` - هيكل الصفحة الرئيسية.
- `css/styles.css` - أنماط العرض والواجهة.
- `js/app.js` - جميع منطق واجهة المستخدم والتكامل مع Firebase.

## كيفية التشغيل

يمكنك فتح `index.html` مباشرة في المتصفح، أو استخدام خادم محلي لعرض المشروع بشكل أفضل.

### تشغيل عبر خادم محلي بسيط

- باستخدام `Live Server` في VS Code
- أو عبر Python:
  - `python -m http.server 8000`
  - ثم افتح `http://localhost:8000`

## ملاحظات

- يستخدم المشروع مكتبات خارجية عبر CDN: FontAwesome، Chart.js، Firebase، و jsPDF.
- تأكد من أن اتصال الإنترنت متاح لتحميل المكتبات.
- بيانات Firebase موجودة بهذا الكود وقد تحتاج إلى تكوين المشروع الخاص بك إذا أردت استبدال قاعدة البيانات.
